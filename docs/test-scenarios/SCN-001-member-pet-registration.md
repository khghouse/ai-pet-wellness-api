# SCN-001: 회원 가입부터 반려견 등록까지

## 목적

신규 사용자가 회원 가입, 로그인, 내 정보 조회를 거쳐 반려견 등록을 완료하는 핵심 API 사용자 흐름을 검증한다.

## 사전 조건

- 테스트 프로필의 데이터베이스와 Redis Testcontainers를 사용한다.
- Flyway로 초기 견종 데이터가 등록되어 있다.

## 시나리오

1. 이메일과 비밀번호로 회원 가입을 요청한다.
2. 동일한 이메일과 비밀번호로 로그인하여 Access Token을 발급받는다.
3. Access Token으로 내 정보 조회를 요청한다.
4. Access Token과 활성 상태 견종을 사용해 반려견 등록을 요청한다.

## 기대 결과

- 회원 가입 응답에 회원 식별자, 이메일, `ACTIVE` 상태가 포함된다.
- 로그인 응답에 비어 있지 않은 Access Token과 Refresh Token이 포함된다.
- 내 정보 조회 응답의 회원 식별자와 이메일, 상태가 회원 가입 응답과 일치한다.
- 반려견 등록 응답에 반려견 식별자, 입력한 반려견 정보, 선택한 견종, `OWNER` 멤버십 역할이 포함된다.
