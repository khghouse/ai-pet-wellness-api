  - **제안:** 미션 참여와 달성은 개방하되, 리워드 수령 시 사용자 본인인증과 반려견 검증 완료를 모두 요구한다.

  - **트레이드오프:** 본인인증 및 반려견 검증 외부 연동, 개인정보 처리와 검증 실패 정책이 필요하다.
