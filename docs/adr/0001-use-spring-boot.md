 # 0001. Spring Boot 사용

## 상태

승인됨

## 배경

이 프로젝트는 Java 기반 API 서버를 구현하기 위한 백엔드 프레임워크가 필요하다.

## 결정

주요 백엔드 프레임워크로 Spring Boot를 사용한다.

## 결과

- 웹 API, 검증, 영속성, 보안, 테스트 등에서 Spring 생태계를 활용할 수 있다.
- 프로젝트 컨벤션은 Spring Boot 애플리케이션 구조와 자연스럽게 맞춰야 한다.
- 프레임워크 관련 결정이 아키텍처나 장기 유지보수에 영향을 준다면 후속 ADR로 기록한다.

## 검토한 대안

- 순수 Java 애플리케이션:
  - 이유: 백엔드 API 프로젝트에 필요한 인프라를 직접 구현해야 하는 부담이 크다.
- 다른 JVM 프레임워크:
  - 이유: 이 프로젝트에서는 Spring Boot가 더 넓은 생태계와 익숙한 컨벤션을 제공한다.
